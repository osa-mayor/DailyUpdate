## 13. [Qwen-AgentWorld: Language World Models for General Agents](https://qwen.ai/blog?id=qwen-agentworld)
**Source**: Qwen Blog | **Category**: Open-Source | **Release Type**: official_update

### 요약
Qwen-AgentWorld는 7개 도메인에 걸쳐 에이전트 환경을 시뮬레이션하는 네이티브 언어 월드 모델입니다. 일반적인 LLM에 사후 적응시키는 방식이 아니라, 지속적 사전 학습(CPT) 단계부터 환경 모델링을 학습 목표로 설정하여 개발되었습니다.

### 핵심 포인트
- CPT → SFT → RL 과정을 통해 환경 모델링을 내재화한 네이티브 언어 월드 모델
- 텍스트 기반 환경(MCP, Search, Terminal, SWE 등)을 포함한 7개 도메인 시뮬레이션
- 단일 모델로 다양한 에이전트 환경을 구현하는 것을 목표로 함

**태그**: Qwen-AgentWorld, World Model, AI Agent

### 원문 설명
Today we release Qwen-AgentWorld, a native language world model that simulates agent environments across seven domains: Native world modeling: environment modeling is the training objective from continual pre-training onward (CPT → SFT → RL), not a post hoc adaptation on top of a general-purpose LLM. Seven domains, one model: a single model simulates text-based (MCP, Search, Terminal, SWE)...
