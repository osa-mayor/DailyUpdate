### 🧑‍💻 개발자 관점
에이전트가 실시간 환경에서 과거 데이터를 기반으로 판단해야 하는 복잡한 워크플로우(예: 로봇 제어, 장기 작업 수행) 설계 시 필수적인 평가 기준을 제공합니다.

- 긴 컨텍스트를 가진 멀티모달 작업에서 정보 손실이 발생하는 지점 분석
- 에이전트의 행동 데이터(Rollouts)를 활용한 미세 조정 파이프라인 구축 테스트

### ⚠️ 리스크/한계

- 에이전트 워크플로우, 액션 및 관찰 공간의 체계적 설계를 통한 Guava 프레임워크 개발

- 소규모 모델을 대상으로 시뮬레이션 데이터를 활용한 지식 증류(Distillation) 실험 설계

- 추상화된 액션 공간이 매우 정밀한 제어가 필요한 작업에서는 한계가 될 수 있음

### 📝 초록 기반 상세 설명
최근 시각-언어 모델을 활용한 에이전트 연구가 활발하지만, 어떤 설계 요소가 효과적인 에이전트 구동을 가능케 하는지는 불분명했습니다. 본 논문은 에이전트 워크플로우, 액션 공간, 관찰 공간을 체계적으로 탐색하여 설계된 'Guava' 프레임워크를 제안합니다. 연구 결과, 반복적인 인지-추론-행동 루프, 의미론적 액션 추상화, 멀티모달 관찰이 핵심 요소임을 밝혀냈습니다. 이를 바탕으로 소규모 모델(4B)에 이러한 능력을 증류(Distillation)하는 파이프라인을 구축했습니다. 실험 결과, 소량의 시뮬레이션 데이터만으로도 거대 모델에 필적하는 성능과 높은 일반화 능력을 확보했습니다.

- KL-regularization 계수 변화에 따른 생성 다양성 및 품질 변화 관찰하기
- RL 학습 시 Discriminator의 표현 공간이 생성 품질에 미치는 영향 분석하기

### ⚠️ 리스크/한계

### 📝 초록 기반 상세 설명

- 다양한 마스크 크기(Small/Medium/Large)에 따른 속도 및 품질 벤치마크 수행
- 제안된 주파수 기반 토큰 선택 전략을 기존 DiT 아키텍처에 이식 테스트
- 편집 영역 경계에서의 아티팩트 발생 여부 확인

### ⚠️ 리스크/한계
- 마스크 경계면에서의 토큰 불일치로 인한 시각적 아티팩트 발생 가능성
- 특정 주파수 기반 전략이 복잡한 질감의 배경에서 성능 저하를 일으킬 위험

### 📝 초록 기반 상세 설명
최근 생성형 AI 모델이 Diffusion Transformer(DiT)로 전환됨에 따라 이미지 편집 작업에서의 지연 시간(Latency) 문제가 심화되고 있습니다. 특히 DiT 모듈은 전체 모델 지연 시간의 상당 부분을 차지하여 실시간 편집 서비스에 걸림돌이 됩니다. 이를 해결하기 위해 입력 데이터에 적응적으로 토큰을 할당하는 HiLo-Token 프레임워크를 제안합니다. 편집 영역은 확장된 마스크를 통해 컨텍스트를 유지하고, 배경 영역은 공간 주파수 기반의 선택 전략과 다운샘플링된 토큰을 혼합하여 효율적으로 처리합니다. 실험 결과, 생성 품질 저하 없이 마스크 크기에 따라 최대 3.13배의 DiT 속도 향상을 달성했습니다.
