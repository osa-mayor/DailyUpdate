## 1. [Mega-ASR: Towards In-the-wild^2 Speech Recognition via Scaling up Real-world Acoustic Simulation](https://huggingface.co/papers/2605.19833)

**arXiv**: https://arxiv.org/abs/2605.19833

**태그**: ASR, Robustness, Data Augmentation, Speech-to-Text, Audio, Benchmark

![Thumbnail](images/Mega-ASR_Towards_In-the-wild^2_Speech_Recognition_via_Scaling_up_Real-world_Acoustic_Simulation_img.jpg)

### 📌 한 줄 요약
복합적인 실생활 소음 환경에서도 강건한 음성 인식을 위해 대규모 시뮬레이션 데이터와 단계적 최적화 기법을 결합한 Mega-ASR 프레임워크 제안

### 🔑 핵심 포인트
- 7가지 음향 현상 및 54가지 시나리오를 포함하는 대규모 합성 데이터셋 'Voices-in-the-Wild-2M' 구축
- 음향적 특징에서 의미적 특징으로 전이되는 단계적 SFT(Acoustic-to-Semantic Progressive SFT) 적용
- WER(Word Error Rate) 기반의 이중 정밀도 정책 최적화(Dual-Granularity WER-Gated Policy Optimization)를 통한 강건성 확보

### 🧑‍💻 개발자 관점
실제 소음이 심한 환경(카페, 거리 등)에서 ASR 모델을 배포할 때 발생하는 성능 저하와 환각 문제를 해결할 수 있는 데이터 및 학습 전략을 제공합니다.

### 🚀 실무 적용 아이디어
- 제공된 Voices-in-the-Wild-2M 데이터셋의 합성 방식 분석 및 자사 데이터에 적용
- 복합 소음 환경에서의 모델 성능 측정을 위한 벤치마크 파이프라인 구축
- SFT 단계에서 음향-의미 간 점진적 학습 스케줄링 실험

### ⚠️ 리스크/한계
- 합성 데이터(Simulation) 기반 학습이 실제 환경의 미세한 물리적 차이를 완벽히 커버하지 못할 가능성
- 복합적인 시나리오 구성 시 데이터 생성 비용 및 복잡도 증가

### 📝 초록 기반 상세 설명
최근 ASR 기술의 발전에도 불구하고, 복합적인 소음이 발생하는 실생활 환경에서는 모델이 음성 정보를 놓치거나 환각(hallucination)을 일으키는 문제가 지속되고 있습니다. 이를 해결하기 위해 연구진은 대규모 합성 데이터 구축과 단계적 최적화를 결합한 Mega-ASR 프레임워크를 제안합니다. 먼저 7가지 음향 현상과 54가지 복합 시나리오를 포함하는 Voices-in-the-Wild-2M 데이터셋을 구축했습니다. 학습 과정에서는 Acoustic-to-Semantic Progressive SFT와 Dual-Granularity WER-Gated Policy Optimization 기법을 적용했습니다. 실험 결과, Mega-ASR는 기존 SOTA 모델 대비 열악한 환경의 벤치마크에서 월등한 성능 향상을 보였으며, 복합 소음 시나리오에서도 높은 강건성을 입증했습니다.

## 3. [Enhancing Train-Free Infinite-Frame Generation for Consistent Long Videos](https://huggingface.co/papers/2605.18233)
**Upvotes**: 77 | **도입 난이도**: 하 | **신뢰도**: 상
**arXiv**: https://arxiv.org/abs/2605.18233

**태그**: Video Generation, Long-form Video, Zero-shot, Consistency, RAG, Video, Inference, Safety

![Thumbnail](images/Enhancing_Train-Free_Infinite-Frame_Generation_for_Consistent_Long_Videos_img.jpg)

### 📌 한 줄 요약
추가 학습 없이 일관성 있는 초장기 비디오 생성을 가능하게 하는 MIGA 프레임워크 제안

### 🔑 핵심 포인트
- 학습 없이 기존 파운데이션 모델을 활용하는 무학습(Train-free) 프레임워크
- 노이즈 스팬 조절을 통한 학습-추론 간 불일치(Training-Inference Gap) 완화
- 자기 성찰 및 장거리 가이드를 결합한 이중 일관성 강화 메커니즘

### 🧑‍💻 개발자 관점
모델 재학습 비용 없이 기존 비디오 생성 모델의 출력 길이를 확장하고 일관성을 확보할 수 있는 실용적인 방법론입니다.

### 🚀 실무 적용 아이디어
- 기존 FIFO-diffusion 등 자기회귀 모델에 MIGA의 노이즈 정렬 로직 적용 테스트
- 다양한 해상도 및 프레임 레이트에서의 장기 일관성 유지 능력 검증
- Self-reflection 메커니즘이 초기 프레임의 품질에 미치는 영향 분석

### ⚠️ 리스크/한계
- 무학습 방식이므로 모델 자체의 근본적인 생성 능력 한계에 종속됨
- 매우 긴 비디오 생성 시 연산 복잡도 및 가이드 프레임 관리 이슈 발생 가능성

### 📝 초록 기반 상세 설명
기존의 프레임 단위 자기회귀 방식은 메모리 효율적이지만, 학습과 추론 간의 불일치 및 장기적 일관성 유지 문제로 인해 긴 비디오 생성에 한계가 있습니다. 이를 해결하기 위해 본 논문은 새로운 무학습(train-free) 방식인 MIGA를 제안합니다. 먼저, 모델에 입력되는 과도한 노이즈 범위를 줄여 학습-추론 간 격차를 해소하는 2단계 정렬 메커니즘을 도입합니다. 또한, 초기 고노이즈 프레임을 수정하는 자기 성찰(self-reflection) 방식과 후기 저노이즈 프레임을 가이드로 활용하는 장거리 프레임 가이드 방식을 결합한 이중 일관성 강화 메커니즘을 제안합니다. 실험 결과, MIGA는 VBench 및 NarrLV 벤치마크에서 최첨단 성능을 달립니 다.

## 4. [You Only Need Minimal RLVR Training: Extrapolating LLMs via Rank-1 Trajectories](https://huggingface.co/papers/2605.21468)
**Upvotes**: 38 | **도입 난이도**: 하 | **신뢰도**: 상
**arXiv**: https://arxiv.org/abs/2605.21468

**태그**: LLM, Reinforcement Learning, Parameter Efficiency, Optimization, Reasoning, Benchmark

![Thumbnail](images/You_Only_Need_Minimal_RLVR_Training_Extrapolating_LLMs_via_Rank-1_Trajectories_img.jpg)

### 📌 한 줄 요약
RLVR 학습 과정의 파라미터 변화가 매우 낮은 차원(Rank-1)임을 발견하여, 극소량의 학습만으로 최종 모델 성능을 예측/생성하는 효율적 기법 제안

### 🔑 핵심 포인트
- RLVR 학습 궤적이 매우 낮은 랭크(Rank-1)를 가지며 선형적으로 변화하는 특성 발견
- RELEX: 짧은 관찰 구간의 랭크-1 서브스페이스를 이용한 선형 회귀 기반 체크포인트 외삽 기법
- 학습 비용을 획기적으로 절감하면서도(최대 1/20 수준) 높은 성능을 유지하는 효율성 입증

### 🧑‍💻 개발자 관점
모델 학습 비용이 막대한 상황에서, 전체 학습을 완료하지 않고도 목표 성능에 도달할 수 있는 '조기 종료 및 예측' 전략의 가능성을 제시합니다.

### 🚀 실무 적용 아이디어
- 현재 진행 중인 RLVR 학습 데이터의 파라미터 델타(delta) 랭크 분석해보기
- 학습 초기 단계의 체크포인트를 이용한 선형 외삽 실험 수행
- 학습 스텝에 따른 성능 향상 곡선과 파라미터 변화량의 상관관계 확인

### ⚠️ 리스크/한계
- 특정 데이터셋이나 특정 모델 구조에 국한된 현상일 가능성
- 학습 초기 단계의 노이즈가 클 경우 외삽의 정확도가 급격히 떨어질 위험

### 📝 초록 기반 상세 설명
LLM의 추론 능력을 높이기 위한 RLVR(Verifiable Rewards) 학습이 대세가 되었으나, 학습 과정에서의 파라미터 궤적(trajectory) 특성에 대한 연구는 부족했습니다. 본 논문은 RLVR 학습 시 발생하는 파라미터 변화량이 매우 낮은 랭크(Rank-1)를 가지며 선형적으로 진화한다는 점을 발견했습니다. 이를 바탕으로 짧은 학습 구간만 관찰하여 미래의 체크포인트를 예측하는 RELEX 기법을 제안합니다. 실험 결과, 전체 학습의 15%만 수행하고도 전체 학습과 대등하거나 더 나은 성능을 달순하는 놀라운 외삽(extrapolation) 능력을 보였습니다. 이는 학습 과정의 노이즈를 제거하고 핵심적인 업데이트 방향만을 추출함으로써 가능했습니다.

## 37. [Lost in the Folds: When Cross-Validation Is Not a Deep Ensemble for Uncertainty Estimation](https://huggingface.co/papers/2605.18329)
**Upvotes**: 1 | **도입 난이도**: 하 | **신뢰도**: 상
**arXiv**: https://arxiv.org/abs/2605.18329

**태그**: Deep Learning, Computer Vision, Uncertainty Estimation, Medical Imaging, Vision, Evaluation

![Thumbnail](images/Lost_in_the_Folds_When_Cross-Validation_Is_Not_a_Deep_Ensemble_for_Uncertainty_Estimation_img.jpg)

### 📌 한 줄 요약
K-fold 교차 검증(CV)과 딥 앙상블(DE)은 불확실성 추정 방식이 다르므로 연구 목적에 따라 엄격히 구분하여 사용해야 합니다.

### 🔑 핵심 포인트
- CV와 DE의 불확실성 생성 원리 차이 규명 (데이터 분할 vs 시드 변동성)
- 목적에 따른 앙상블 전략 제안 (신뢰성/실패 탐지: DE, 모호성 모델링: CV)
- nnU-Net 파이프라인 내에서 DE를 쉽게 적용할 수 있는 경량 수정 방식 제공

### 🧑‍💻 개발자 관점
모델의 불확실성을 측정할 때 단순히 CV를 사용하는 것이 딥 앙상블과 동일한 효과를 주지 않음을 인지하고, 목적에 맞는 앙상블 전략을 설계해야 합니다.

### 🚀 실무 적용 아이디어
- 현재 사용 중인 CV 기반 앙상블이 DE와 동일한 성능/불확실성 특성을 보이는지 비교 실험
- 실패 탐지(Failure Detection)가 목적일 경우 CV 대신 DE 방식으로 전환
- 제공된 nnU-Net 수정 방식을 적용하여 DE 파이프라인 구축 테스트

### ⚠️ 리스크/한계
- 데이터셋의 특성에 따라 CV와 DE의 성능 역전 가능성 존재
- DE는 동일 데이터셋을 반복 학습하므로 CV보다 연산 비용이 낮을 수 있으나 과적합 위험이 있음

### 📝 초록 기반 상세 설명
의료 영상 분할 분야에서는 앙상블 간의 불일치를 에피스테믹 불확실성(epistemic uncertainty)의 대리 지표로 자주 사용합니다. 그러나 많은 연구가 K-fold 교차 검증(CV)을 수행하면서 이를 '딥 앙상블(DE)'로 혼용하여 용어와 구현의 불일치를 야기합니다. 본 연구는 동일한 설정에서 5-fold CV와 5-member DE를 비교 분석하여 불확실성 측정 능력을 검증했습니다. 실험 결과, DE는 정확도 유지와 함께 캘리브레이션 및 실패 탐지에 강점을 보인 반면, CV는 데이터 노출 차이로 인해 레이블 간 모호성을 반영하는 경향이 있었습니다. 따라서 신뢰성 중심의 작업에는 DE를, 데이터의 모호성 모델링에는 CV를 선택하는 것이 적절합니다.

## 38. [Learning from Language Feedback via Variational Policy Distillation](https://huggingface.co/papers/2605.15113)
**Upvotes**: 1 | **도입 난이도**: 상 | **신뢰도**: 상
**arXiv**: https://arxiv.org/abs/2605.15113

**태그**: Reinforcement Learning, LLM Reasoning, Policy Distillation, Self-Improvement, Reasoning, Vision, Evaluation, Distillation

![Thumbnail](images/Learning_from_Language_Feedback_via_Variational_Policy_Distillation_img.jpg)

### 📌 한 줄 요약
언어 피드백을 활용한 교사-학생 모델의 동적 상호 진화를 통해 복잡한 추론 태스크의 학습 병목을 해결하는 프레임워크

### 🔑 핵심 포인트
- 언어 피드백 학습을 변분 EM 프레임워크로 정형화하여 교사와 학생 모델의 동시 진화 구현
- 고정된 교사 모델의 한계를 극복하기 위해 적응형 신뢰 영역(Trust-region) 기반의 교사 모델 업데이트 도입
- 희소한 결과 보상을 밀집된 토큰 수준의 분포 가이드로 변환하여 학습 효율 극대화

### 🧑‍💻 개발자 관점
LLM이 스스로 생성한 결과물에 대해 피드백을 주고 학습하는 'Self-improvement' 루프를 구축할 때, 교사 모델의 성능 정체 문제를 해결할 수 있는 방법론을 제시합니다.

### 🚀 실무 적용 아이디어
- 특정 도메인(수학, 코딩) 데이터셋에 대해 기존 Self-distillation과 VPD의 수렴 속도 비교 실험
- 교사 모델의 업데이트 강도(Trust-region 파라미터)에 따른 학생 모델의 학습 안정성 테스트
- 피드백의 질이 낮은 'Cold-start' 상황에서의 프레임워크 강건성 검증

### ⚠️ 리스크/한계
- 교사 모델과 학생 모델이 동시에 학습됨에 따라 발생할 수 있는 학습 불안정성 및 모드 붕괴(Mode collapse) 위험
- 복잡한 수식 계산 및 엄격한 논리 구조가 필요한 태스크에서의 성능 한계 가능성

### 📝 초록 기반 상세 설명
복잡한 추론 작업에서 검증 가능한 보상(RLVR)은 희소한 신호로 인해 탐색의 어려움을 겪습니다. 기존의 언어 피드백 기반 자기 증류(Self-distillation) 방식은 고정된 교사 모델에 의존하여 학생 모델의 성장에 따라 학습 효율이 정체되는 한계가 있습니다. 이를 해결하기 위해 본 논문은 언어 피드백 학습을 변분 EM(Variational EM) 문제로 정형화한 VPD(Variational Policy Distillation)를 제안합니다. E-step에서는 신뢰 영역 업데이트를 통해 교사 모델을 정교화하고, M-step에서는 학생 모델이 이 밀집된 분포 가이드를 학습합니다. 실험 결과, VPD는 과학적 추론 및 코드 생성 태스크에서 기존 RLVR 및 자기 증식 방식보다 우수한 성능을 보였습니다.
