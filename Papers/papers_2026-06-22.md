## 1. [Moebius: 0.2B Lightweight Image Inpainting Framework with 10B-Level Performance](https://huggingface.co/papers/2606.19195)
**Upvotes**: 112 | **도입 난이도**: 중 | **신뢰도**: 상
**arXiv**: https://arxiv.org/abs/2606.19195

**태그**: Image Inpainting, Model Compression, Diffusion Models, Efficiency, Vision, Benchmark, Inference, Distillation, Optimization, Safety

![Thumbnail](images/Moebius_0.2B_Lightweight_Image_Inpainting_Framework_with_10B-Level_Performance_img.jpg)

### 📌 한 줄 요약
0.2B 규모의 초경량 파라미터로 10B급 성능을 구현한 고효율 이미지 인페인팅 프레임워크

### 🔑 핵심 포인트
- LλMI(Local-λ Mix Interaction) 블록을 통한 효율적인 공간 및 글로벌 정보 요약
- 잠재 공간 내에서 수행되는 적응형 다중 입도 증류(Multi-granularity Distillation) 전략
- 거대 모델(FLUX.1-Fill-Dev) 대비 압도적인 추론 속도 및 파라미터 효율성 확보

### 🧑‍💻 개발자 관점
엣지 디바이스나 실시간 서비스 환경에서 고성능 인페인팅 기능을 저비용·고속으로 구현할 수 있는 실질적인 가이드를 제공합니다.

### 🚀 실무 적용 아이디어
- 제안된 LλMI 블록 구조를 기존 경량 Diffusion 모델에 이식하여 성능 변화 확인
- 다양한 해상도 환경에서 잠재 공간 기반 증류 전략의 안정성 테스트
- 실제 프로덕션 환경에서의 추론 속도(Latency)와 메모리 점유율 벤치마크

### ⚠️ 리스크/한계
- 극단적인 압축으로 인해 특정 복잡한 텍스처에서 디테일 손실 가능성
- 증류(Distillation) 과정에서의 하이퍼파라미터 튜닝 난이도

### 📝 초록 기반 상세 설명
최근 10B 이상의 거대 모델이 이미지 인페인팅 성능을 높였으나, 막대한 연산 비용으로 인해 실무 배포에 어려움이 있습니다. 모델을 압축할 경우 발생하는 표현력 저하 문제를 해결하기 위해 Moebius 프레임워크를 제안합니다. 핵심 기술로 공간 컨텍스트와 글로벌 의미를 고정 크기 행렬로 요약하는 LλMI 블록을 도입하여 파라미터를 획기적으로 줄였습니다. 여기에 잠재 공간 내에서 다중 그래디언트 손실을 조절하는 적응형 다중 입도 증류(Multi-granularity Distillation) 전략을 결합했습니다. 실험 결과, Moebius는 기존 거대 모델 대비 2% 미만의 파라미터만 사용하면서도 대등하거나 더 뛰어난 생성 품질을 보여주었습니다.

- 멀티 언어 코딩 작업 투입 전, 모델의 언어 간 성능 격차(Disparity) 측정하기

## 4. [Playful Agentic Robot Learning](https://huggingface.co/papers/2606.19419)
**Upvotes**: 43 | **도입 난이도**: 중 | **신뢰도**: 상
**arXiv**: https://arxiv.org/abs/2606.19419

**태그**: Robot Learning, Agentic Workflow, Code-as-Policy, Skill Library, Agent, Inference, Distillation

![Thumbnail](images/Playful_Agentic_Robot_Learning_img.jpg)

### 📌 한 줄 요약
자율적인 '놀이(Play)'를 통해 로봇의 재사용 가능한 코드 스킬 라이브러리를 사전 구축하는 에이전트 학습 프레임워크

### 🔑 핵심 포인트
- 자율적 놀이(Self-directed play)를 통한 지속적인 스킬 습득 메커니즘 제안
- 실패 진단 및 반복 실행을 통해 성공적인 코드를 스킬 라이브러리로 정제하는 RATs 프레임워크
- 학습된 코드를 컨텍스트에 주입하는 것만으로 타 에이전트의 성능을 높이는 높은 재사용성

### 🧑‍💻 개발자 관점
모델 전체를 재학습하지 않고도 코드 스킬 라이브러리(RAG 방식과 유사)를 통해 로봇의 작업 수행 능력을 즉각적으로 확장할 수 있는 실용적인 접근법을 제시합니다.

### 🚀 실무 적용 아이디어
- 코드 기반 정책(Code-as-Policy) 환경에서 스킬 라이브러리 구축 및 검색 로직 구현 실험
- 에이전트의 자율적 과제 생성(Task Generation) 알고리즘의 다양성 검증
- 학습된 코드 스킬이 새로운 환경(Zero-shot)에서 얼마나 견고하게 작동하는지 테스트

### ⚠️ 리스크/한계
- 놀이 단계에서 생성된 스킬이 실제 복잡한 작업에 유효하지 않을 가능성(Skill Gap)
- 라이브러리 규모가 커질 경우 검색 및 컨텍스트 관리의 복잡도 증가

### 📝 초록 기반 상세 설명
기존의 에이전트 기반 로봇 시스템은 작업 중심적이라 명시적인 지시가 있을 때만 기술을 습득하는 한계가 있습니다. 본 논문은 작업 수행 전 단계로 자율적인 '놀이'를 통해 지속적인 스킬 학습을 수행하는 Playful Agentic Robot Learning을 제안합니다. 이를 위해 RATs(Robotics Agent Teams)를 도입하여, 에이전트가 스스로 탐색적 과제를 생성하고, 코드를 실행하며, 실패를 진단하고 성공적인 코드를 스킬 라이브러리에 저장하도록 설계했습니다. 테스트 시에는 이 라이브러리에서 관련 스킬을 검색하여 새로운 작업에 재사용합니다. 실험 결과, 놀이를 통해 학습된 스킬은 기존 방식 대비 다운스트림 작업 성능을 유의미하게 향상시켰습니다. 또한, 학습된 스킬은 모델 파인튜닝 없이 컨텍스트에 삽입하는 것만으로도 다른 에이전트의 성능을 높이는 범용성을 보여주었습니다.
