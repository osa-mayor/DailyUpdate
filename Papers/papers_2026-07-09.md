## 4. [Hierarchical Sparse Attention Done Right: Toward Infinite Context Modeling](https://huggingface.co/papers/2607.02980)
**Upvotes**: 35 | **도입 난이도**: 중 | **신뢰도**: 상
**arXiv**: https://arxiv.org/abs/2607.02980

**태그**: LLM, Long-Context, Sparse Attention, Efficiency, RAG, Evaluation

![Thumbnail](images/Hierarchical_Sparse_Attention_Done_Right_Toward_Infinite_Context_Modeling_img.jpg)

### 📌 한 줄 요약
학습 시보다 64배 더 긴 문맥을 처리할 수 있는 계층적 희소 어텐션(HiLS) 메커니즘 제안

### 🔑 핵심 포인트
- LM 손실을 활용한 엔드투엔드 청크 선택 및 검색 최적화
- 계층적 구조를 통한 효율적인 정보 추출 및 융합 메커니즘
- 기존 모델을 경량 지속 사전 학습(continued pretraining)을 통해 HiLS로 전환 가능

### 🧑‍💻 개발자 관점
모델의 연산 효율을 유지하면서도 기존 모델의 한계를 넘어선 초장거리 문맥(Ultra-long context)을 처리할 수 있는 실질적인 방법론을 제시합니다.

### 🚀 실무 적용 아이디어
- 기존 풀 어텐션 모델에 HiLS를 적용하는 경량 지속 사전 학습 실험
- 다양한 문맥 길이에서의 검색 정확도 및 성능 유지력 벤치마크
- KV 캐시 메모리 절감 효과와 추론 속도 간의 트레이드오프 분석

### ⚠️ 리스크/한계
- 학습 시 문맥 길이에 따른 검색 정확도 유지 여부 검증 필요
- 특정 도메인 데이터에 대한 과적합 가능성

### 📝 초록 기반 상세 설명
기존 LLM의 긴 문맥 처리는 연산 비용의 이차적 증가와 길이 외삽(extrapolation) 문제로 인해 한계가 있습니다. 기존의 청크 단위 희소 어텐션 방식은 부정확한 청크 선택 문제로 인해 전체 어텐션의 성능을 따라가지 못했습니다. 본 논문은 언어 모델링 손실(LM loss)을 통해 청크 선택을 엔드투엔드로 학습하는 계층적 랜드마크 희소(HiLS) 어텐션을 제안합니다. HiLS는 쿼리가 검색된 각 청크로부터 정보를 추출한 뒤 검색 점수에 따라 이를 융합하는 계층적 구조를 가집니다. 실험 결과, HiLS는 학습 시 문맥 길이보다 64배 더 긴 문맥에서도 높은 정확도를 유지하며 기존 풀 어텐션 모델의 성능을 유지하면서도 초장거리 문맥을 처리할 수 있음을 입증했습니다.

## 37. [SiamJEPA: On the Role of Siamese Student Encoders in JEPA](https://huggingface.co/papers/2607.04044)

### 📌 한 줄 요약
Siamese 구조를 도입하여 JEPA의 표현 학습 효율과 정규화 성능을 극대화한 자기주도 학습 프레임워크

### 🔑 핵심 포인트
- Siamese 학생 인코더와 EMA 교사 네트워크를 결합한 새로운 JEPA 구조 제안

### 🧑‍💻 개발자 관점
데이터 효율성이 중요한 환경에서 적은 학습 비용으로도 고성능 비전 모델을 구축할 수 있는 새로운 설계 지침을 제공합니다.

- 학습 예산(Budget) 제한 조건에서 Siamese 구조의 수렴 속도 측정 실험
- 다양한 데이터셋 규모에 따른 Siamese 구조의 정규화 효과 검증

### ⚠️ 리스크/한계
- Siamese 구조 도입에 따른 추가적인 연산 비용 및 메모리 사용량 증가 가능성

### 📝 초록 기반 상세 설명
최근 픽셀 재구성이 아닌 잠재 임베딩을 예측하는 JEPA 방식이 자기주도 학습의 유망한 프레임워크로 주목받고 있습니다. 기존 I-JEPA나 V-JEPA는 단일 인코더를 사용하는 경우가 많아, 뇌 모방 학습 구조인 Siamese 구조의 잠재적 이점을 충분히 활용하지 못했습니다. 본 논문은 Siamese 학생 인코더와 EMA 교사 네트워크를 결합한 SiamJEPA를 제안합니다. 실험 결과, Siamese 구조는 JEPA 목적 함수에 대한 효과적인 정규화 역할을 수행하며 학습 초기 단계의 속도와 표현 분리도를 높였습니다. 결과적으로 SiamJEPA는 제한된 예산 내에서 기존 단일 인코더 모델을 능가하며, MAE보다 높은 선형 프로빙 정확도를 달ек했습니다.

## 38. [VIBE: Voice-Induced open-ended Bias Evaluation for Large Audio-Language Models via Real-World Speech](https://huggingface.co/papers/2604.17248)
**Upvotes**: 0 | **도입 난이도**: 중 | **신뢰도**: 상
**arXiv**: https://arxiv.org/abs/2604.17248

**태그**: LALM, Bias-Evaluation, Speech-Processing, Fairness, RAG, Audio, Benchmark, Evaluation

![Thumbnail](images/VIBE_Voice-Induced_open-ended_Bias_Evaluation_for_Large_Audio-Language_Models_via_Real-World_Speech_img.jpg)

### 📌 한 줄 요약
실제 음성 데이터를 활용해 LALM의 생성형 편향을 측정하는 개방형 평가 프레임워크 VIBE 제안

### 🔑 핵심 포인트
- 실제 음성 데이터를 활용한 개방형(Open-ended) 편향 평가 프레임워크 VIBE 제안
- 객관식(MCQ) 방식의 한계를 넘어 고정관념이 자연스럽게 드러나는 평가 환경 구축
- 성별 및 억양 단서가 LALM의 생성 결과에 미치는 체계적 편향성 입증

### 🧑‍💻 개발자 관점
LALM 기반 서비스 개발 시, 단순 정답 맞추기가 아닌 실제 사용자 음성 입력 시 발생할 수 있는 잠재적 편향성을 검증하는 데 중요합니다.

### 🚀 실무 적용 아이디어
- 현재 개발 중인 LALM 모델에 VIBE 프레임워크를 적용하여 성별/억양 편향성 테스트
- 개방형 생성 태스크(예: 추천, 요약)에서 특정 속성이 결과값에 미치는 영향 분석
- 다양한 억양 데이터셋을 확보하여 모델의 강건성(Robustness) 검증

### ⚠️ 리스크/한계
- 개방형 평가 특성상 정량적 비교를 위한 표준화된 지표 설정이 어려울 수 있음
- 사용된 데이터셋의 인구통계학적 특성에 따라 결과가 국한될 가능성

### 📝 초록 기반 상세 설명
최근 대형 오디오-언어 모델(LALM)이 일상 애플리케이션에 통합되고 있으나, 생성형 편향에 대한 연구는 부족한 실정입니다. 기존의 공정성 벤치마크는 합성 음성과 객관식 문제(MCQ)에 의존하여 편향을 단편적으로만 측정하는 한계가 있습니다. 본 논문은 실제 사람이 녹음한 음성을 사용하여 개인화 추천과 같은 개방형 태스크를 통해 생성형 편향을 평가하는 VIBE 프레임워크를 제안합니다. VIBE는 미리 정의된 선택지 없이 고정관념이 자연스럽게 발현되도록 유도하며, 새로운 태스크로의 확장성도 높습니다. 12종의 최신 LALM을 평가한 결과, 성별과 억양 단서가 통계적으로 유의미한 분포 변화를 일으키는 체계적 편향이 발견되었습니다.
