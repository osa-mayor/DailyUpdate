## 1. [UniVidX: A Unified Multimodal Framework for Versatile Video Generation via Diffusion Priors](https://huggingface.co/papers/2605.00658)
**Upvotes**: 66 | **도입 난이도**: 중 | **신뢰도**: 상
**arXiv**: https://arxiv.org/abs/2605.00658

**태그**: Video Generation, Diffusion Models, Multimodal, Computer Vision, RAG, Video, Safety

![Thumbnail](images/UniVidX_A_Unified_Multimodal_Framework_for_Versatile_Video_Generation_via_Diffusion_Priors_img.jpg)

### 📌 한 줄 요약
단일 비디오 확산 모델(VDM)을 활용하여 다양한 멀티모달 비디오 생성 작업을 통합 처리하는 범용 프레임워크

### 🔑 핵심 포인트
- Stochastic Condition Masking (SCM): 학습 시 모달리티를 무작위로 조건 또는 타겟으로 분할하여 양방향 조건부 생성을 가능케 함
- Decoupled Gated LoRA (DGL): 모달리티별 전용 LoRA를 도입하여 VDM의 기본 지식을 유지하면서 특정 모달리티 생성 성능을 최적화
- Cross-Modal Self-Attention (CMSA): 모달리티 간 Key/Value를 공유하여 정보 교환 및 정렬을 촉진하는 구조

### 🧑‍💻 개발자 관점
하나의 모델로 비디오 편집, 레이어 분리, 속성 생성 등 다양한 태스크를 통합 관리할 수 있어 모델 관리 비용과 데이터 효율성을 크게 높일 수 있습니다.

### 🚀 실무 적용 아이디어
- 제공된 SCM 기법을 기존 Diffusion 모델 학습 파이프라인에 적용하여 양방향 생성 테스트
- LoRA를 활용한 멀티모달 데이터셋(RGB-Depth 등)의 통합 학습 실험
- 적은 수의 비디오 데이터(1,000개 미만)를 활용한 Zero-shot/Few-shot 일반화 성능 검증

### ⚠️ 리스크/한계
- 모달리티 수가 급격히 늘어날 경우 CMSA 구조의 연산 복잡도 증가 가능성
- 매우 복잡한 고해상도 비디오 생성 시 단일 모델의 용량 한계 발생 가능성

### 📝 초록 기반 상세 설명
최근 비디오 확산 모델(VDM)을 다양한 그래픽 작업에 활용하려는 시도가 있으나, 기존 방식은 각 작업마다 별도의 모델을 학습시켜 입력-출력 매핑이 고정되고 모달리티 간 상관관계 학습이 어렵다는 문제가 있었습니다. 이를 해결하기 위해 본 논문은 VDM의 사전 학습된 지식을 활용하는 통합 프레임워크인 UniVidX를 제안합니다. SCM을 통해 조건부 생성과 타겟 생성을 유연하게 전환하고, DGL로 모달리티별 특성을 보존하며, CMSA를 통해 모달리티 간 정렬을 유도합니다. 이를 통해 RGB 영상과 intrinsic map(albedo, normal 등) 또는 RGBA 레이어 생성과 같은 복잡한 작업을 하나의 모델로 수행합니다. 실험 결과, 적은 데이터셋으로도 기존 SOTA 모델들과 경쟁력 있는 성능을 보이며 뛰어난 일반화 능력을 입증했습니다.

![Thumbnail](images/Prox-E_Fine-Grained_3D_Shape_Editing_via_Primitive-Based_Abstractions_img.jpg)

### 📌 한 줄 요약
기하학적 프리미티브 추상화를 통해 객체의 정체성을 유지하면서 정교한 3D 구조 편집을 가능하게 하는 프레임워크

### 🔑 핵심 포인트
- 프리미티브 기반의 명시적 기하학적 추상화 도입
- VLM을 활용한 텍스트 기반의 정교한 구조 편집 제어
- 학습이 필요 없는(Training-free) 효율적인 3D 편집 파이프라인

### 🧑‍💻 개발자 관점
기존 2D 기반 3D 편집의 고질적 문제인 '형태 왜곡'을 구조적 추상화로 해결하여, 실무적인 3D 에셋 수정 워크플로우에 적용 가능합니다.

### 🚀 실무 적용 아이디어
- 복잡한 메쉬 구조가 프리미티브로 얼마나 정확하게 추상화되는지 테스트
- VLM의 편집 지시가 실제 3D 구조 변화로 변환되는 정밀도 검증
- 기존 2D-to-3D 파이프라인과의 생성 속도 및 품질 비교

### ⚠️ 리스크/한계
- 복잡도가 매우 높은 비정형 객체의 경우 프리미티브 추상화 오차 발생 가능성
- 프리미티브 개수에 따른 편집의 세밀도와 연산 비용 간의 트레이드오프

### 📝 초록 기반 상세 설명
최근 2D 기반 3D 편집 기술이 발전했으나, 객체의 전체적인 정체성을 유지하면서 국소적인 구조를 변경하는 정교한 편집에는 한계가 있었습니다. 이를 해결하기 위해 본 논문은 프레임워크인 Prox-E를 제안합니다. 이 방식은 3D 형상을 압축된 기하학적 프리미티브 세트로 추상화한 뒤, 사전 학습된 VLM을 통해 프리미티브 수준의 편집을 수행합니다. 이렇게 생성된 구조적 편집 정보는 3D 생성 모델을 가이드하여, 기존 형태를 보존하면서도 정밀한 국소 수정을 가능하게 합니다. 실험 결과, 기존 2D 기반 및 학습 기반 방식보다 정체성 보존, 품질, 지시 이행 측면에서 우수한 성능을 보였습니다.

## 5. [Learning while Deploying: Fleet-Scale Reinforcement Learning for Generalist Robot Policies](https://huggingface.co/papers/2605.00416)

**arXiv**: https://arxiv.org/abs/2605.00416

**태그**: Reinforcement Learning, Robot Policy, VLA, Fleet Learning, Continual Learning, RAG, Vision

![Thumbnail](images/Learning_while_Deploying_Fleet-Scale_Reinforcement_Learning_for_Generalist_Robot_Policies_img.jpg)

### 📌 한 줄 요약
배포 중 수집되는 플릿(Fleet) 데이터를 활용해 범용 로봇 정책을 지속적으로 개선하는 오프라인-온라인 강화학습 프레임워크

### 🔑 핵심 포인트
- 배포-경험 공유-정책 개선-재배포로 이어지는 폐쇄 루프(Closed-loop) 플릿 학습 프레임워크 제안
- 이질적이고 희소한 플릿 데이터를 안정적으로 학습하기 위한 DIVL 및 QAM 알고리즘 도입
- 사전 학습된 VLA 정책을 실세계의 다양한 작업에 맞춰 지속적으로 최적화하는 능력 검증

### 🧑‍💻 개발자 관점
실제 환경에 배포된 로봇이 새로운 상황을 만날 때마다 데이터를 수집하고 이를 다시 모델 성능 향상으로 연결하는 자동화된 파이프라인 구축 모델을 제시합니다.

### 🚀 실무 적용 아이디어
- 사전 학습된 VLA 모델에 소규모 로봇 환경에서 데이터 수집 및 재학습 루프 구현 테스트
- 인간 개입(Human intervention) 데이터가 포함된 데이터셋에서의 정책 안정성 실험
- Sparse reward 환경에서의 DIVL/QAM 알고리즘 성능 비교 실험

### ⚠️ 리스크/한계
- 실제 로봇 배포 시 발생하는 데이터 편향이 정책의 특정 상황 과적합을 유발할 위험
- 플릿 규모가 커질수록 데이터 정제 및 학습 안정성 유지 비용 증가

기존의 결합형 오디오-비디오 생성 모델은 모든 모달리티를 하나의 어텐션 메커니즘으로 묶어 처리함으로써 고수준 의미와 저수준 디테일이 복잡하게 얽히는 문제를 가집니다. 이는 서로 다른 렌더링 과정을 거치는 오디오와 비디오 특성을 고려할 때 효율성과 품질을 저해할 수 있습니다. 본 논문은 고수준 교차 모달 모델링은 공유 백본에서 수행하고, 저수준 정밀화는 모달리티별 디코더를 사용하는 Talker-T2AV를 제안합니다. 공유된 오토레그레시브 언어 모델이 통합된 패치 토큰 공간에서 오디오와 비디오를 공동 추론하며, 두 개의 경량 확산 트랜스포머 헤드가 각 모달리티의 잠재 변수를 생성합니다. 실험 결과, 제안 모델은 기존 이중 분기 방식보다 입술 동기화(lip-sync), 비디오 및 오디오 품질 면에서 우수한 성능을 보였습니다.

### ⚠️ 리스크/한계
