## 2. [Does Your Reasoning Model Implicitly Know When to Stop Thinking?](https://huggingface.co/papers/2602.08354)

**태그**: LLM, Reasoning, Efficiency, Sampling, Reinforcement Learning, Benchmark, Inference

![Thumbnail](images/Does_Your_Reasoning_Model_Implicitly_Know_When_to_Stop_Thinking_img.jpg)

### 📌 한 줄 요약
LLM은 스스로 언제 추론을 멈춰야 하는지 알고 있지만, 기존 샘플링 방식 때문에 활용되지 못하고 있으며, 이를 활용하는 SAGE라는 새로운 샘플링 방식을 통해 성능과 효율을 개선할 수 있다.

### 🔑 핵심 포인트
- LRM이 스스로 추론 중단 시점을 알고 있다는 사실 발견
- 효율적인 추론을 위한 새로운 샘플링 방식 SAGE 제안
- SAGE를 강화 학습에 통합하여 수학 벤치마크에서 성능 및 효율성 향상

### 🧑‍💻 개발자 관점
LLM의 추론 과정을 개선하여 비용을 절감하고 응답 속도를 향상시킬 수 있으며, 특히 실시간 추론이 중요한 서비스에 적용할 수 있다.

### 🚀 실무 적용 아이디어
- SAGE 샘플링 방식을 기존 LLM 추론 파이프라인에 적용해보기
- 자체 데이터셋에 SAGE-RL을 적용하여 성능 개선 효과 확인해보기
- 추론 단계별 시간과 정확도 간의 상관관계를 분석하여 최적의 중단 시점 파악하기

### ⚠️ 리스크/한계
- SAGE가 특정 유형의 추론 문제에만 효과적일 수 있음
- SAGE-RL의 학습 과정이 복잡하고 자원 소모적일 수 있음

### 📝 초록 기반 상세 설명
최근 대규모 추론 모델(LRM)은 긴 CoT(Chain-of-Thought)를 통해 복잡한 추론 작업에서 뛰어난 성능을 보여주지만, 종종 불필요한 중복으로 인해 계산 효율성이 떨어지고 실시간 애플리케이션에 지연을 초래한다. 기존 연구에 따르면 긴 추론 체인이 항상 정확도와 관련이 있는 것은 아니며 오히려 해로울 수도 있다. 본 연구에서는 LRM이 언제 추론을 멈춰야 하는지 스스로 알고 있음을 밝혀냈지만, 기존 샘플링 방식에 의해 가려져 있다. 이를 바탕으로 효율적인 추론 잠재력을 활용하는 새로운 샘플링 방식 SAGE(Self-Aware Guided Efficient Reasoning)를 제안한다. 또한, SAGE를 그룹 기반 강화 학습(SAGE-RL)에 통합하여 추론 정확도와 효율성을 크게 향상시켰다.

## 6. [EgoPush: Learning End-to-End Egocentric Multi-Object Rearrangement for Mobile Robots](https://huggingface.co/papers/2602.18071)

**태그**: Robotics, Reinforcement Learning, Egocentric Vision, Object Rearrangement, Vision, Video, Distillation

![Thumbnail](images/EgoPush_Learning_End-to-End_Egocentric_Multi-Object_Rearrangement_for_Mobile_Robots_img.jpg)

### 📌 한 줄 요약
EgoPush는 로봇이 1인칭 시점에서 여러 물체를 재배치하는 것을 학습하는 새로운 프레임워크로, 시뮬레이션에서 학습된 정책을 실제 로봇에 바로 적용할 수 있습니다.

### 🔑 핵심 포인트
- 1인칭 시점 기반의 비-파지(non-prehensile) 물체 재배치 학습 프레임워크 EgoPush 제안
- 전체 상태 추정 없이 객체 중심 잠재 공간을 활용하여 복잡한 환경에서의 재배치 문제 해결
- 시뮬레이션에서 학습된 정책을 실제 로봇에 제로샷으로 전이하는 데 성공

### 🧑‍💻 개발자 관점
로봇의 시각 정보를 활용하여 물체 재배치 작업을 자동화하는 데 유용하며, 특히 GPS나 정밀한 위치 정보 없이 로봇이 자율적으로 작업을 수행해야 하는 환경에서 활용 가능성이 높습니다. SLAM 등의 기존 방식이 어려운 환경에 적용을 고려해볼 수 있습니다.

### 🚀 실무 적용 아이디어
- EgoPush 프레임워크를 사용하여 특정 로봇 환경에 맞는 물체 재배치 정책 학습
- 물체 중심 잠재 공간 설계 방식을 참고하여 다른 로봇 작업에 적용
- 시뮬레이션-실제 간 전이 성능을 높이기 위한 추가적인 연구 진행

### ⚠️ 리스크/한계
- 복잡한 환경이나 다양한 물체 종류에 대한 일반화 성능 검증 필요
- 시뮬레이션 환경과 실제 환경 간의 격차로 인한 성능 저하 가능성 존재

### 📝 초록 기반 상세 설명
인간은 1인칭 시점을 통해 복잡한 환경에서 물체를 재배치하고, 전체 좌표 없이도 가려진 부분을 탐색합니다. 이러한 능력에 영감을 받아, 본 논문에서는 단일 1인칭 카메라를 사용하여 모바일 로봇이 장기간에 걸쳐 여러 물체를 재배치하는 문제를 연구합니다. EgoPush는 명시적인 전체 상태 추정 없이도 1인칭 시점 기반의 지각 주도 재배치를 가능하게 하는 정책 학습 프레임워크입니다. 물체의 절대적인 자세 대신 상대적인 공간 관계를 인코딩하는 객체 중심 잠재 공간을 설계하고, 부분적으로 관찰된 학생 정책으로 증류합니다. 시뮬레이션에서 EgoPush는 기존 엔드 투 엔드 RL 방법보다 성능이 우수하며, 실제 로봇에서도 제로샷 전이가 가능함을 입증합니다.

## 8. [VidEoMT: Your ViT is Secretly Also a Video Segmentation Model](https://huggingface.co/papers/2602.17807)
**Upvotes**: 2 | **도입 난이도**: 중 | **신뢰도**: 상
**arXiv**: https://arxiv.org/abs/2602.17807

**태그**: Vision, Segmentation, Transformer, Video

![Thumbnail](images/VidEoMT_Your_ViT_is_Secretly_Also_a_Video_Segmentation_Model_img.jpg)

### 📌 한 줄 요약
복잡한 트래킹 모듈 없이 비전 트랜스포머(ViT) 인코더만으로 고속 비디오 분할이 가능한 Video Encoder-only Mask Transformer (VidEoMT) 모델을 제안하여 비디오 분할 속도를 5~10배 향상시킴.

### 🔑 핵심 포인트
- 트래킹 모듈 없는 ViT 기반 비디오 분할 모델 VidEoMT 제안
- 쿼리 전파 및 쿼리 융합을 통한 효율적인 시간적 정보 처리
- 기존 모델 대비 5~10배 빠른 속도와 경쟁력 있는 정확도 달성

### 🧑‍💻 개발자 관점
비디오 분석 파이프라인에서 복잡한 트래킹 모듈 없이 고성능 비디오 분할을 구현할 수 있게 되어, 실시간 분석이나 고해상도 비디오 처리 시 효율성을 크게 높일 수 있습니다.

### 🚀 실무 적용 아이디어
- VidEoMT 코드를 다운로드하여 기존 비디오 분할 모델과 성능 비교
- 쿼리 전파 및 쿼리 융합 메커니즘을 다른 비디오 처리 작업에 적용
- VidEoMT를 사용하여 실시간 비디오 분석 시스템 구축

### ⚠️ 리스크/한계
- 대규모 사전 학습된 ViT 모델에 대한 의존성
- 특정 유형의 비디오 (예: 급격한 장면 전환)에서 성능 저하 가능성

### 📝 초록 기반 상세 설명
기존 온라인 비디오 분할 모델은 프레임별 분할기와 복잡한 트래킹 모듈을 결합하여 사용했지만, 이는 구조적 복잡성과 계산 비용을 증가시키는 문제점이 있었습니다. 최근 연구에 따르면 대규모 사전 학습된 ViT 인코더는 별도의 모듈 없이도 정확한 이미지 분할이 가능합니다. 이러한 점에 착안하여 본 논문에서는 트래킹 모듈이 필요 없는 간단한 인코더 전용 비디오 분할 모델인 VidEoMT를 제안합니다. VidEoMT는 쿼리 전파 메커니즘을 통해 이전 프레임의 쿼리를 재사용하여 시간적 모델링을 가능하게 하고, 시간적으로 독립적인 학습된 쿼리와 결합하여 새로운 콘텐츠에 대한 적응성을 높입니다. 결과적으로 VidEoMT는 트래커의 이점을 유지하면서도 복잡성을 줄여 정확도를 유지하면서 최대 160 FPS로 실행되어 5~10배 빠른 속도를 달성합니다.
