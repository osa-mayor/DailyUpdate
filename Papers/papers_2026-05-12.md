## 1. [Mean Mode Screaming: Mean--Variance Split Residuals for 1000-Layer Diffusion Transformers](https://huggingface.co/papers/2605.06169)
**Upvotes**: 87 | **도입 난이도**: 중 | **신뢰도**: 상
**arXiv**: https://arxiv.org/abs/2605.06169

**태그**: Diffusion Transformer, Deep Learning, Model Scaling, Optimization

![Thumbnail](images/Mean_Mode_Screaming_Mean--Variance_Split_Residuals_for_1000-Layer_Diffusion_Transformers_img.jpg)

### 📌 한 줄 요약
초거대 심층 Diffusion Transformer의 표현력 붕괴(Mean-dominated collapse) 문제를 해결하기 위한 새로운 잔차 연결(Residual) 구조 제안

### 🔑 핵심 포인트
- 심층 DiT에서 발생하는 평균값 편향 붕괴 현상(MMS) 규명
- 평균 성분과 중심 변동 성분을 분리하여 업데이트하는 MV-Split Residuals 제안
- 400층 및 1000층 규모의 초심층 모델에서의 학습 안정성 확보

### 🧑‍💻 개발자 관점
모델의 깊이가 깊어질수록 발생하는 수렴 실패 문제를 구조적으로 해결하여, 초거대 규모의 생성 모델 설계 시 안정성을 높일 수 있습니다.

### 🚀 실무 적용 아이디어
- 심층 Transformer 모델 학습 시 Loss가 갑자기 급감하거나 표현력이 뭉개지는지 모니터링
- 기존 LayerScale 대신 MV-Split 방식의 잔차 구조 적용 테스트
- 모델 깊이 증가에 따른 그래디언트 성분(Mean vs Centered) 분석 실험

### ⚠️ 리스크/한계
- 매우 깊은 층을 다루는 특수 상황에 최적화된 기법으로, 일반적인 중소규모 모델에서는 효과가 미미할 수 있음
- 평균과 변동 성분을 분리하는 추가적인 연산 비용 발생 가능성

### 📝 초록 기반 상세 설명
Diffusion Transformer(DiT)를 수백 층 이상으로 확장할 때, 토큰 표현이 균일해지며 변동성이 사라지는 'Mean Mode Screaming(MMS)' 현상이 발생합니다. 연구진은 이 현상이 잔차 연결(Residual)을 통해 평균값에 편향된 그래디언트가 주입되어 네트워크가 붕괴되는 것임을 밝혀냈습니다. 이를 해결하기 위해 평균 성분과 중심 변동 성분을 분리하여 업데이트하는 'Mean-Variance Split(MV-Split) Residuals' 방식을 제안합니다. 실험 결과, MV-Split은 400층 규모의 DiT에서 붕괴 없이 안정적인 학습을 가능하게 했으며, 1000층 규모에서도 학습 안정성을 입증했습니다. 이는 기존의 LayerScale과 같은 게이팅 방식보다 심층 네트워크에서 더 우수한 성능을 보입니다.

Appearance Expert는 분리된 운동-미학 미세 조정 전략을 통해 참조 이미지의 정체성을 유지하며 일관된 비디오를 합성합니다. 연구진은 이 작업을 위한 대규모 데이터셋과 새로운 평가 프로토콜을 구축하였으며, 실험 결과 SOTA 성능을 달성했습니다.

## 3. [Flow-OPD: On-Policy Distillation for Flow Matching Models](https://huggingface.co/papers/2605.08063)
**Upvotes**: 73 | **도입 난이도**: 상 | **신뢰도**: 상
**arXiv**: https://arxiv.org/abs/2605.08063

**태그**: Generative AI, Flow Matching, Reinforcement Learning, Knowledge Distillation, Image Generation, RAG, Vision, Evaluation, Distillation, Safety

![Thumbnail](images/Flow-OPD_On-Policy_Distillation_for_Flow_Matching_Models_img.jpg)

### 📌 한 줄 요약
멀티태스크 정렬 시 발생하는 성능 충돌(Seesaw effect)을 해결하기 위해 On-Policy Distillation을 Flow Matching 모델에 적용한 통합 사후 학습 프레임워크

### 🔑 핵심 포인트
- GRPO 기반의 전문가 교사 모델 학습 및 온폴리 증류(On-Policy Distillation)를 통한 지식 통합 전략
- 태스크 간 간섭을 최소화하기 위한 3단계 오케스트레이션(샘플링, 라우팅, 밀집 궤적 감독) 프로세스
- 생성 품질 유지를 위한 Manifold Anchor Regularization(MAR) 도입

### 🧑‍💻 개발자 관점
다양한 요구사항(OCR, 미적 품질 등)을 동시에 만족시켜야 하는 범용 이미지 생성 모델 개발 시, 지표 간 충돌 문제를 해결하는 실질적인 방법론을 제시합니다.

### 🚀 실무 적용 아이디어
- GRPO와 같은 강화학습 알고리즘을 Flow Matching 모델에 적용할 때의 학습 안정성 테스트
- 다양한 태스크(텍스트 렌더링 vs 미적 품질)를 동시에 학습시킬 때의 지표 간 상관관계 분석
- 제안된 MAR 기법을 기존 Diffusion 모델의 정렬 과정에 적용하여 품질 저하 여부 확인

### ⚠️ 리스크/한계
- 교사 모델의 품질에 대한 의존도가 높으며, 학습 복잡도가 매우 높음
- 온폴리 샘플링 과정에서의 높은 계산 비용 및 학습 시간 소요

### 📝 초록 기반 상세 설명
기존 Flow Matching 기반 텍스트-이미지 모델은 스칼라 보상으로 인한 희소성 및 이질적 목적 함수 간의 간섭 문제로 인해 특정 지표가 상승하면 다른 지표가 하락하는 '시소 효과'와 보상 해킹 문제를 겪습니다. 이를 해결하기 위해 본 논문은 LLM의 성공 사례를 차용한 Flow-OPD 프레임워크를 제안합니다. 먼저 GRPO를 통해 각 도메인에 특화된 전문가 교사 모델을 학습시킨 후, 온폴리 샘플링과 태스크 라우팅을 통해 이들의 지식을 하나의 학생 모델로 통합합니다. 또한, 생성 품질 저하를 막기 위해 태스크에 무관한 교사 모델을 활용하는 Manifold Anchor Regularization(MAR)을 도입했습니다. 실험 결과, Stable Diffusion 3.5 Medium을 기반으로 GenEval 및 OCR 정확도를 대폭 향상시키며 교사 모델을 능가하는 성능을 달성했습니다.

## 5. [Listwise Policy Optimization: Group-based RLVR as Target-Projection on the LLM Response Simplex](https://huggingface.co/papers/2605.06139)

### 🔑 핵심 포인트
- 기존 그룹 기반 RLVR의 수학적 구조를 '응답 심플렉스 상의 타겟 투영'으로 재해석

- 단조적 개선(Monotonic improvement)과 제약된 그래디언트를 통한 학습 안정성 및 다양성 확보

### 🧑‍💻 개발자 관점

### 🚀 실무 적용 아이디어
- 기존 PPO나 GRPO와 같은 그룹 기반 RL 알고리즘과 LPO의 학습 곡선 및 응답 다양성 비교 실험
- 다양한 Divergence(KL 등) 설정에 따른 LPO의 수렴 속도 및 성능 변화 테스트
- 추론 태스크(Math, Code) 외에 일반 대화 태스크에서의 응답 다양성 유지 능력 검증

### ⚠️ 리스크/한계
- 심플렉스 상의 타겟 정의가 복잡한 태스크에서 여전히 유효한지 검증 필요

- 추출된 세만틱 델타의 품질이 모델의 편집 성능에 직접적인 영향을 미침

### 📝 초록 기반 상세 설명
기존의 예시 기반 이미지 편집 방식은 동일한 편집 의미를 가진 두 쌍의 이미지(pair-of-pairs)가 필요하여 대규모 데이터 구축이 어려웠습니다. 본 논문은 텍스트 가이드 없이 단일 이미지 쌍만으로 학습 가능한 Delta-Adapter를 제안합니다. 사전 학습된 비전 인코더를 통해 소스-타겟 이미지 간의 '세만틱 델타(semantic delta)'를 추출하고, 이를 Perceiver 기반 어댑터를 통해 편집 모델에 주입합니다. 타겟 이미지를 직접 노출하지 않는 구조 덕분에 단일 쌍만으로도 학습이 가능하며, 세만틱 델타 일관성 손실(consistency loss)을 통해 변환의 정확도를 높였습니다. 실험 결과, 기존 방식보다 편집 정확도와 콘텐츠 일관성 면에서 우수하며 미학습 작업에 대한 일반화 성능도 뛰어남을 입증했습니다.
