## 5. [How Taalas “prints” LLM onto a chip?](https://www.anuragk.com/blog/posts/Taalas.html)

### 📰 원문 기사 요약
Taalas는 Llama 3.1 8B 모델(3/6 bit 양자화)을 ASIC 칩에 내장하여 초당 17,000 토큰의 추론 속도를 달성했다. 이는 기존 GPU 기반 시스템보다 10배 저렴한 비용과 전력 소모, 10배 빠른 속도를 제공한다고 주장한다. 칩은 고정 기능 ASIC으로, CD-ROM처럼 하나의 모델만 저장하고 수정할 수 없다. Llama 3.1 8B는 32개의 레이어로 구성되어 있으며, 각 레이어는 모델의 '지식'인 대규모 가중치 행렬을 포함한다.

### 💬 Hacker News 토론 요약
한 사용자는 자신의 박사 논문 주제와 유사하다며 Taalas의 접근 방식을 흥미롭게 평가했다(Modern Gate Array Design). 다른 사용자는 8B개의 계수를 53B개의 트랜지스터에 패킹하는 효율성에 주목하며, 계수당 6.5개의 트랜지스터가 사용된 점을 지적했다.

### 📌 종합 요약
신생 기업 Taalas가 Llama 3.1 8B 모델을 ASIC 칩에 '하드와이어링'하여 GPU 기반 시스템보다 훨씬 저렴하고 빠른 추론 속도를 달성했다고 주장하며 화제다. 이 기술은 LLM을 칩에 직접 '인쇄'하는 듯한 방식으로, 기존 소프트웨어 개발 방식과는 다른 접근 방식을 보여준다.

### 🔎 종합 핵심 포인트
- Taalas는 LLM을 ASIC 칩에 직접 '하드와이어링'하는 방식으로 기존 GPU 기반 추론 시스템 대비 성능 향상 및 비용 절감을 달성했다.
- 토론 참여자들은 Taalas의 접근 방식의 독창성과 효율성에 주목하며, 구체적인 트랜지스터 사용량 등을 분석했다.
- 이 기술은 특정 모델에 최적화된 고정 기능 칩의 가능성을 보여주며, 향후 LLM 하드웨어 개발 방향에 시사하는 바가 크다.
