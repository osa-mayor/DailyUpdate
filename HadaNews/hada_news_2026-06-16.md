사용자가 입력한 SQL 스키마를 브라우저 내에서 즉시 ER 다이어그램으로 시각화해 주는 무료 도구가 공개되었습니다. PostgreSQL, MySQL, SQLite, SQL Server 등 다양한 DBMS의 CREATE TABLE 및 ALTER TABLE DDL 문법을 지원합니다. 데이터가 외부 서버로 전송되지 않고 클라이언트 측 브라우저에서만 처리되므로 보안성이 높으며, 기본 키, 외래 키, unique, not-null 등 주요 제약 조건을 정확히 파싱하여 관계를 시각화합니다.
